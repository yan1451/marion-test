# Bank Robbery Slot

Projeto desenvolvido como teste tecnico para a MERION.

## Requisitos

- Node.js 18+ (recomendado)
- npm

## Como executar

1. Instale as dependencias:

```bash
npm install
```

2. Execute em ambiente de desenvolvimento:

```bash
npm run dev
```

3. Abra o endereco exibido no terminal (normalmente `http://localhost:5173`).

## Scripts disponiveis

- `npm run dev`: inicia o servidor de desenvolvimento com Vite
- `npm run start`: alias para `npm run dev`
- `npm run lint`: executa validacao de lint
- `npm run build`: executa lint, compilacao TypeScript e build de producao
